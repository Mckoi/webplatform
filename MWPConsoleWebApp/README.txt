
Mckoi Web Platform User Interface web application (MWPUIApp)
============================================================

This repository contains the source code for the Mckoi Web Platform
developer and administrator interface. This web application, once deployed
in a Mckoi Web Platform installation, provides various tools for creating
and managing web applications. The tools provided include a console,
editor, compiler, and data administration.
